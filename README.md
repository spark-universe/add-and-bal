# 발주 & 광고 설정 훈련 사이트

수강생 교육용 웹사이트의 **기초 틀(스캐폴딩)**입니다. 순수 HTML/CSS/JS로 되어 있어 별도 설치·빌드 없이 파일을 더블클릭하면 바로 열립니다.

## 실행 방법
- `index.html` 을 브라우저로 열면 **사용자 화면**이 시작됩니다.
- 사이드바 하단 링크 또는 `admin/index.html` 로 **어드민 화면**에 접근합니다.

## 폴더 구조
```
Add_and_Bal/
├─ index.html              사용자 · 메인
├─ basic-settings.html     사용자 · 기본 설정 연습하기
├─ ad-settings.html        사용자 · 광고 설정하기
├─ order-setup.html        사용자 · 발주 연습 세팅
├─ order-practice.html     사용자 · 발주 연습하기
├─ admin/
│  ├─ index.html           어드민 · 메인(대시보드)
│  ├─ registrants.html     어드민 · 가입자 관리(승인)
│  ├─ users.html           어드민 · 사용자 관리(승급/자료)
│  ├─ review.html          어드민 · 자료 검수
│  └─ results.html         어드민 · 결과 관리
├─ css/style.css           공통 디자인
├─ js/layout.js            공통 사이드바 생성
└─ _backup_숫자야구.html    (기존에 있던 무관한 파일 백업)
```

## 현재 상태 (기초 틀)
- 사용자/어드민 10개 화면의 레이아웃·사이드바·네비게이션 완성
- 테이블·통계 카드·폼·자료제출 카드 등 UI 컴포넌트 배치
- 데이터는 모두 비어 있는 상태(예시 행은 주석 처리)

## 다음 단계 (미구현)
- 회원가입/로그인 (성명·전화번호·구글 이메일, 중복 방지)
- 자료 제출 업로드 및 검수 연동
- 발주 연습 로직 (임시 주문 생성, 사기 주문·역마진·정보누락 케이스)
- 백엔드/DB 연동, 진도·참조 가능 영역 잠금 처리
- 결과 집계 및 성적 산출
