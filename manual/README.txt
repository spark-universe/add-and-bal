Shopify 실무 매뉴얼 · 3단원 스토어 초기 작업
================================================

■ 구성
  index.html   … 매뉴얼 본문 (웹페이지)
  images/      … 본문에 들어가는 스크린샷 이미지 폴더

■ 여는 방법
  index.html 와 images 폴더는 반드시 같은 위치에 함께 두세요.
  index.html 를 더블클릭하면 브라우저에서 매뉴얼이 열립니다.

■ 웹사이트에 올리는 방법
  1) index.html 와 images 폴더를 통째로 서버에 업로드하면 그대로 게시됩니다.
  2) 워드프레스·카페24 등 CMS를 쓰는 경우: images 폴더의 이미지를 미디어에
     업로드한 뒤, index.html 에서 각 챕터(<section>) HTML을 복사해 붙여 넣고
     이미지 경로만 교체하면 됩니다.

■ 참고
  · '준비 중' 챕터(소싱 앱 · 스파크 · 상품 업로드 · 컬렉션 · 파비콘/로고 · 푸터)는
    자료가 준비되면 채워 넣을 수 있도록 자리를 비워 두었습니다.
  · '이미지 준비 중' 으로 표시된 4곳은 원본 문서에서 이미지가 누락된 위치입니다.
